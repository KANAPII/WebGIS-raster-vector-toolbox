# 地理信息分析工具（WebGIS）

本项目为作者本科毕业设计，属于短期开发项目。目前仍在开发中，毕业后计划完全开源。

当前仓库提供的是**前端程序包**，支持本地运行，但由于缺少后端服务，部分功能暂时无法正常使用。
开源后会完整提供后端python代码。

## 当前支持功能

### 支持的数据格式
- Shapefile（`.shp`）
- GeoJSON
- TIFF 栅格文件（`.tiff` / `.tif`）

### 栅格数据处理工具
- 坡度分析（后端）
- 像元值筛选（后端）
- 按掩膜提取像元（后端）
- 按掩膜去除像元（后端）
- 像元赋值（后端）
- 栅格镶嵌至新栅格（后端）

### 矢量数据处理工具
- 缓冲区分析（前端）
- 栅格化矢量数据
- 相交（前端）
- 合并（前端）
- 擦除（前端）
- 对称差（前端）

### 属性表
- 可以浏览矢量文件的属性表，并进行修改、查找字段、新建字段等基础操作，并支持了字段计算器
  
## 项目特点
- 适合轻量级地理信息分析工作
- 图层显示采用**快照（Snapshot）**方式，有效降低大数据量带来的页面卡顿
- 默认坐标系：**Web 墨卡托投影（EPSG:3857）** 和 **WGS84（EPSG:4326）**
- 默认集成在线地图（图源为 OpenStreetMap），因依赖后端服务，目前本地运行时不可见。开源后可自行修改 API 地址。

## 本地运行方法

1. 确保已安装 `serve` 工具（若未安装，可执行 `npm install -g serve`）
2. 进入项目 `dist` 目录，执行以下命令：

```bash
serve -s dist -p 你的端口
```


3. 打开浏览器访问 http://localhost:你的端口

## 注意事项

- 当前版本无法使用需要后端支持的功能（包括在线地图等）
- 项目仍在开发完善中，部分功能可能存在不稳定情况
- 本地运行仅支持前端数据加载与处理
